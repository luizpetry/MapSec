# MapSec Roadmap

Future features and improvements planned for the project.

---

## New Plugins

- [ ] **Shodan** — IoT/device search on exposed services
- [ ] **Whois** — domain/IP registration info
- [ ] **SSL/TLS Check** — certificate analysis, protocols, vulnerabilities
- [ ] **HTTP Headers** — security headers check (CSP, HSTS, X-Frame-Options, etc.)
- [ ] **Banner Grabbing** — collect service banners from open ports
- [ ] **Image Metadata (EXIF)** — extract and analyze EXIF data from images (camera, GPS, software, timestamps)

## Export & Reports

- [ ] **HTML Report** — professional styled report for sharing
- [ ] **PDF Export** — printable format for clients/team
- [ ] **CSV Export** — spreadsheet-compatible output

## Quality & DevOps

- [ ] **Unit Tests** (pytest) — tests for plugins, engine, models
- [ ] **CI/CD** — GitHub Actions for automated tests + build
- [ ] **PyPI Publishing** — `pip install mapsec`

## Features

- [ ] **Scan Profiles** — quick / full / stealth presets
- [ ] **Scan History** — save and compare results over time
- [ ] **CVE Lookup** — vulnerability database integration (NVD API)
- [ ] **Network Topology** — visualize discovered hosts/connections
